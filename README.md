# YOYOW Improvement Proposals  YOYOW治理与发展提案

YOYOW 治理与发展提案（YOYOW Improvement Proposals，以下简称YIP）是由社区发起讨论的议题在取得社区共识后的正式版本，YIP通常由议题相关人员撰写，提交编辑审查后公示。

YIP作为记录YOYOW设计决策、收集社区意见、提议新功能的主要机制，对YOYOW整个生态的健康发展发挥重要作用。
